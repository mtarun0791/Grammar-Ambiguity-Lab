# Grammar-Ambiguity-Lab
Grammar Ambiguity Lab is an interactive tool to analyze ambiguity in Context-Free Grammars (CFGs). It lets users input grammars and strings, generate multiple parse trees and derivations, and determine if a grammar is ambiguous, helping visualize key Theory of Computation concepts.
 Overview

Grammar Ambiguity Lab is an interactive tool developed to analyze ambiguity in Context-Free Grammars (CFGs). A grammar is ambiguous if a single string can be derived in more than one way, producing multiple parse trees. This project helps visualize and understand this concept through practical experimentation.

 Objectives
Understand ambiguity in CFGs
Generate multiple derivations of a string
Identify ambiguous grammars
Visualize parsing concepts effectively
Features
Input custom grammar rules
Test strings for ambiguity
Generate leftmost and rightmost derivations
Detect multiple parse trees
Simple and user-friendly interface
 Concepts Covered
Context-Free Grammar (CFG)
Parse Trees
